# 🇬🇧 English

## About Project

STOREX is a modern mobile application developed with Flutter for stock management, customer shopping operations, and admin control.

The application includes two user roles:

* Customer Panel
* Admin Panel

Customers can browse products, add them to cart, place orders, and review order history.

Admins can manage products, monitor stock levels, review customer orders, and track critical inventory.

---

## Main Features

### Customer Features

* Register / Login
* Browse products by category
* Add to cart
* Payment method selection
* Complete purchase
* Order history

### Admin Features

* Secure admin login
* Product stock management
* Increase / decrease stock
* Delete products
* Customer list
* Order management
* Critical stock warning system

### UI / UX

* Light Mode / Dark Mode
* Clean modern design
* Responsive layout
* Animated buttons

# 🇹🇷 Türkçe

## Proje Hakkında

STOREX, Flutter ile geliştirilmiş modern bir stok takip ve alışveriş yönetim uygulamasıdır.

Uygulama içerisinde iki farklı kullanıcı paneli bulunmaktadır:

* Müşteri Paneli
* Admin Paneli

Müşteriler ürünleri görüntüleyebilir, sepete ekleyebilir, sipariş verebilir ve sipariş geçmişlerini görüntüleyebilir.

Admin tarafında ise stok yönetimi, sipariş kontrolü, müşteri görüntüleme ve kritik stok takibi yapılabilmektedir.
